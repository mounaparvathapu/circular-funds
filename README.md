
Circular Funds is a decentralized finance (DeFi) application designed to modernize peer-to-peer (P2P) rotating savings and credit associations (ROSCAs), commonly known as "Chit Funds". By leveraging blockchain technology, this project provides a trustless, transparent, and automated framework for community-based saving and lending.  Project OverviewIn a traditional Chit Fund, members contribute a fixed amount to a pool periodically, and one member receives the lump sum each cycle. Circular Funds digitizes this practice by removing the need for a central foreman, thereby reducing administrative costs and the risk of fraud.  Key FeaturesAutomated Payouts: Smart contracts manage the collection of contributions and distribute the "pot" to the winner of each cycle automatically.  Decentralized Escrow: Funds are held securely in a smart contract rather than by an individual or central authority.  Transparent Ledger: Every transaction, contribution, and payout is recorded on-chain, providing a 100% auditable trail.  Bidding System: Integrated logic handles competitive bidding for cycles, allowing users to access liquidity when needed.  Technical ImplementationSmart Contracts: Developed using Solidity to handle the core logic of fund pooling and distribution.  Frontend: Built with React.js to provide an intuitive and responsive user interface.  Web3 Integration: Uses Ethers.js or Web3.js to facilitate seamless interaction between the browser and the Ethereum blockchain.  Development Framework: Utilizes Hardhat for compiling, deploying, and testing the decentralized application.
